any' :: (a -> Bool) -> [a] -> Bool #

Descripción

Devuelve true si todos los elementos de la lista cumplen una condición

Ejemplos

>>> any' (<10) [1,2,3,4]

>>> any' (<10) [10, 20, 30, 5]
True

>>> any' (>10) [1,2, 3, 4]
False
