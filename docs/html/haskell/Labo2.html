mapx :: t -> [t -> b] -> [b] #

Descripción

Devuelve una lista de valores resultado de aplicar a un valor un listado de funciones. Todos los tipos de las funciones de la lista deben ser iguales

Ejemplos

>>> mapx 10 [even, odd]
